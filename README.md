# Install Node.js
https://nodejs.org/en/download/current

## Instructions to run the backend
```bash
cd backend
```

```bash
npm install
```

```bash
npm run dev
```

## Instructions to run the frontend
```bash
cd frontend
```

```bash
npm install
```

```bash
npm run start
```

## Visit the following URL to see the application running
http://localhost:5173/

## Backend API
